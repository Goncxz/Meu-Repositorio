Meus Conhecimentos

HTML e CSS

Possuo conhecimento e habilidades com HTML que é uma linguagem de marcação e CSS que é um mecanismo para adicionar estilisação a uma página web aplicado diretamente nas tags HTML, utilizo as mesmas para efetuar todos os meus projetos, utilizo tags semanticas e prezo pela organização dos código para melhor leitura e entendimento do mesmo.

JavaScript

Possuo conhecimento e habilidaes com JAVASCRIPT que é uma linguagem de programação interpretada e estruturada de script em alto nível com tipagem dinâmica e multiparadigma, utilizo para estilizar meus projetos, adicionando interações com o usuário, facilidades na leitura da pagina e proporcionando melhor experiência para o usuário.

React

É uma biblioteca usada com a linguagem de programação JavaScript no desenvolvimento Front-end. Sendo uma biblioteca, o React.js é um conjunto de funções padronizadas. Os desenvolvedores a utilizam para construir uma aplicação. Neste caso específico, a utilização se dá no desenvolvimento de interfaces de usuário UI.

NodeJs

Possuo conhecimento e habilidaes com NODE JS q com o Node.js é possível criar aplicações Javascript para rodar como uma aplicação standalone em uma máquina, não dependendo de um browser para a execução, como estamos acostumados. e que permite a execução de códigos JavaScript fora de um navegador web.

Git e Github

Possuo conhecimento e habilidades com o programa Github que é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git, o mesmo permite que programadores utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos disponiveis.

Figma

Figma é um editor online de design gráfico focado na criação de interfaces gráficas e experiência de usuário com ênfase colaborativa. A ferramenta permite que uma equipe de designers tenha acesso simultâneo e trabalhe em cima de um mesmo projeto ao mesmo tempo, facilitando a vida de times em trabalho remoto.
